## 📰 Introdução 

InfoPulse é um site de notícias de tecnologia inspirado no TechCrunch. Nesse projeto, desenvolvi apenas o front-end da página inicial, com foco em design e experiência do usuário.

Me preocupei em fazer uma design atraente, com elementos bem posicionados, e em proporcionar uma boa experiência para o usuário (UX), adicionando interatividade por meio da manipulação do DOM com JavaScript. Além disso, implementei responsividade com classes bootstrap e media queries para proporcionar uma boa visualização em qualquer dispositivo. Segue abaixo as principais seções do site:

- Nav com links da página inicial, botão de pesquisa e menu que abre offcanvas
- Página inicial(hero), com cards de nóticias em destaque e ao lado as principais noticias
- Áreas com lista de últimas notícias, mais populares e notícias por categoria
- Responsividade com breakpoints definidos e baseados no bootstrap

Como projeto é 100% front-end, não há back-end, banco de dados ou persistência de dados, toda a interatividade é feita via JavaScript no lado do cliente.
